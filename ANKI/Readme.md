# Создание фильтрованных колод в AnkiDroid

**Фильтрованная колода** — это временная, динамическая «витрина», которая забирает карточки из ваших основных колод по заданным правилам. Сами карточки физически остаются в своих исходных колодах.

---

## Способ 1: Быстрый (Настраиваемое обучение)

Идеально для стандартных задач:

1. На главном экране нажмите на **название колоды**
2. Нажмите **три точки (⋮)** в правом верхнем углу
3. Выберите **«Настраиваемое обучение»** (Custom Study)
4. Выберите сценарий:
   - **Увеличить лимит на сегодня** — учить больше карточек
   - **Повторить забытые карточки** — для подготовки к экзамену
   - **Учиться заранее** — карточки, которые должны появиться позже
   - **Учиться по тегу/состоянию** — выбрать конкретный тег
5. Настройте параметры и нажмите **ОК**
6. На главном экране появится временная колода

## Способ 2: Продвинутый (Свой уникальный поиск)

### Шаг 1: Создаем колоду через Поиск

1. Нажмите на значок **Лупы (🔍)** (Поиск)
2. Введите запрос:

**Примеры запросов:**

- `tag:экзамен` — все карточки с тегом "экзамен"
- `deck:Английский is:due` — готовые к повторению из колоды "Английский"
- `rated:1:1` — карточки, на которые ответили "Снова" за последний день
- `is:new tag:сложный` — новые карточки с тегом "сложный"
- `prop:ivl>=30` — карточки с интервалом ≥30 дней
- `added:7` — добавленные за последнюю неделю

3. Нажмите **три точки (⋮)** в правом верхнем углу
4. Выберите **«Создать фильтрованную колоду»** (Build Filtered Deck)

### Шаг 2: Настраиваем параметры

**Имя (Name):** Дайте понятное имя (например, "Подготовка к экзамену")

**Поиск (Search):** Ваш запрос из Шага 1 (можно отредактировать)

**Лимит (Limit):** Максимальное количество карточек (например, 50 или 9999)

**Порядок (Order):**

- **Oldest seen first** (Самые старые первыми) — для повторения забытого
- **Random** (Случайный) — для перемешивания
- **Newest seen first** (Новейшие первыми)
- **Ascending intervals** (По возрастанию интервалов)

**Шаги (Steps):** Оставьте как есть (обычно `1 10`)

**Перепланировать карточки (Reschedule cards...):**

- ✅ **Галочка стоит** (по умолчанию) — ответы считаются за полноценные повторения, интервалы меняются
- ❌ **Галочка снята** — карточки просто "покрутятся" и вернутся в исходные колоды без изменения расписания (для быстрого прогона перед экзаменом)

Нажмите **ОК** (или «Создать»)

---

## Как правильно УДАЛЯТЬ фильтрованные колоды

**⚠️ ВАЖНО:** Нельзя просто удалить фильтрованную колоду нажатием "Удалить"!

**Правильный алгоритм:**

1. На главном экране нажмите на фильтрованную колоду
2. Нажмите **три точки (⋮)** в правом верхнем углу
3. Выберите **«Очистить»** (Empty)
   - Все карточки вернутся в свои родные колоды
   - Сохранится их расписание
   - Фильтрованная колода станет пустой
4. Теперь нажмите на пустую колоду → **три точки (⋮)** → **«Удалить»** (Delete)

---

## Золотые правила фильтрованных колод

❌ **Никогда не добавляйте новые карточки прямо в фильтрованную колоду**

- Если добавите карточку через "+" внутри неё, она удалится навсегда при очистке
- Добавляйте новые карточки только в обычные колоды

⚠️ **Не приостанавливайте (Suspend) карточки внутри фильтрованной колоды**

- При очистке они вернутся в активное состояние

✅ **Можно иметь несколько фильтрованных колод одновременно**

- Например: одна для "Забытых за неделю", другая для "Подготовки к тесту"
- Они не будут мешать друг другу

🔄 **Синхронизация работает отлично**

- Фильтрованные колоды синхронизируются с AnkiWeb и десктопным Anki
- Созданная на ПК колода появится на телефоне после синхронизации

## Полезные примеры использования

**1. Повторить всё забытое за неделю:**

- Поиск: `rated:7:1`
- Порядок: Oldest seen first
- Перепланировать: ✅

**2. Подготовиться к экзамену по конкретному тегу:**

- Поиск: `tag:экзамен_математика`
- Лимит: 9999
- Перепланировать: ❌ (чтобы не испортить расписание)

**3. Выучить новые карточки из определённой колоды:**

- Поиск: `deck:Английский is:new`
- Лимит: 20
- Порядок: Random

**4. Повторить карточки с большим интервалом:**

- Поиск: `prop:ivl>=90`
- Порядок: Ascending intervals
- Перепланировать: ✅

## Синтаксис поиска (шпаргалка)

**Теги:**

- `tag:имя` — карточки с тегом
- `tag:*имя*` — карточки с тегом, содержащим "имя"
- `-tag:имя` — карточки БЕЗ тега

**Колоды:**

- `deck:имя` — карточки из колоды
- `deck:имя*` — карточки из колоды и подколод

**Состояние:**

- `is:due` — готовы к повторению сегодня
- `is:new` — новые карточки
- `is:learning` — в процессе обучения
- `is:review` — на повторении
- `is:suspended` — приостановленные
- `is:buried` — отложенные

**Время:**

- `rated:1` — оценены за последний день
- `rated:7:1` — оценены "Снова" за 7 дней
- `added:7` — добавлены за 7 дней
- `edited:1` — отредактированы за 1 день

**Свойства:**

- `prop:ivl>=30` — интервал ≥30 дней
- `prop:reps<5` — менее 5 повторений
- `prop:lapses>2` — более 2 провалов

**Комбинирование:**

- `tag:сложный is:due` — готовые карточки с тегом "сложный"
- `deck:Английский -tag:легкий` — из колоды, но без тега "легкий"
- `(tag:экзамен OR tag:важное) is:new` — новые с любым из тегов
