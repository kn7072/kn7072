set in |set ɪn| 1) Наступать, начинаться и, вероятно, продолжаться (о погоде, состоянии, процессе) 

Panic set in when the fire alarm went off.
Паника началась, когда сработала пожарная сигнализация.

####

Infection can set in if the wound isn't cleaned.
Инфекция может начаться, если рану не очистить.

####

Darkness set in as we walked through the forest.
Темнота наступила, пока мы шли через лес.

####

The rain set in and didn't stop for days.
Начался дождь и не прекращался несколько дней.

####

Winter has set in early this year.
Зима в этом году наступила рано.

####
