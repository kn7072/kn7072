📂 Что находится внутри папки Anki2

Зайдя в папку Anki2, вы увидите отдельную подпапку для каждого вашего профиля (например, User 1 по умолчанию). Внутри папки профиля хранятся самые важные файлы:

**collection.anki2**: Главный файл базы данных. Здесь хранятся все ваши колоды, карточки, заметки и настройки

. Это основной файл для резервного копирования.

**collection.media**: Папка с медиафайлами. Сюда Anki сохраняет все изображения, аудио и видео, которые вы добавляете в карточки
.
backups: Папка, где Anki автоматически хранит резервные копии вашей коллекции

💡 Важный совет
⚠️ Никогда не перемещайте, не копируйте и не изменяйте файл collection.anki2 или папку collection.media вручную, пока открыт Anki. Это может привести к повреждению вашей коллекции
Лучше всего закрыть программу перед тем, как работать с этими файлами.
