meta_d - поля для описание к странице с драйверами.
meta_k - поле для ключевых слов драйверов

Они заполняются однотипно:
1) meta_k : "скачать,драйверы,бесплатно,на высокой скорости,dell,inspiron,3521,windows 7,64 bit" - это пример заполнения для модели ноутбука Dell Inspiron 3521 с драйверами под ОС Windows 7 64 bit
Здесь общими для всех моделей являются "скачать,драйверы,бесплатно,на высокой скорости,dell". Остальное зависит от серии, модели и версии ОС.

2) meta_d : "Скачать бесплатно драйверы на высокой скорости для Dell Inspiron 3521 [Windows 7 64-bit]". Тут соответственно меняется только строка "Dell Inspiron 3521 [Windows 7 64-bit]".
